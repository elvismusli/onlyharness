# Монетизация OnlyHarness и конкурентный ландшафт

Дата: июль 2026. Статус: рабочий документ для обсуждения с партнёрами.
Связанные документы: [custdev-brief-2026-07.md](custdev-brief-2026-07.md), [verified-catalog-2026-07.md](verified-catalog-2026-07.md).

## TL;DR

1. Рынок уже платит создателям агентских инструментов, но только там, где продаётся **исполнение** (Apify: топы $10k+/мес), а не файлы (Agensi/PromptBase: медиана <$50-200/мес).
2. Прямого конкурента с нашей связкой «версионированный харнес + evals/gate + agent-native покупка + оплата по результату» **нет ни у кого**.
3. Платёжные рельсы созрели: x402 Foundation под Linux Foundation (Coinbase, Cloudflare, Stripe, AWS, Google, Visa), 165M+ транзакций, но средний чек $0.30 — микроплатежи работают для API-вызовов, покупка харнесов будет в диапазоне $1–50.
4. Стратегия: доверие бесплатно → подписки на обновления и bounties → платные endpoints через x402 → B2B приватные реестры.

## 1. Модели монетизации

### M1. Платный endpoint (harness-as-a-service) — защищено от кражи by design

Автор публикует бесплатную тонкую обёртку (клиент), «мозг» — промпты и логика — живёт на сервере как MCP/API. Оплата per-call.

- **Момент оплаты:** в момент вызова. `hh run` → HTTP 402 → кошелёк агента платит USDC → ответ.
- **Защита:** красть нечего, файл — клиент.
- **Наша роль:** хостинг «мозгов» + платёжный гейтвей. Take rate 10–20% с вызова.
- **Прецедент:** MCPize делает ровно это для MCP (80/20, hosting+Stripe+x402), Apify — для скраперов (80/20, $1M+/мес выплат разработчикам).

### M2. Подписка на поток обновлений

Файл дёшев или бесплатен; платишь за канал обновлений и совместимость (боль №2 кастдева: «что сломалось при добавлении конфига»).

- **Момент оплаты:** при pull новой версии / месячная подписка.
- **Защита:** украденная копия протухает — не получает фиксы совместимости.
- **Ценовой диапазон:** $5–20/мес за харнес или бандл автора (бенчмарк MCPize: $9–49/мес).

### M3. Оплата по результату (gate-эскроу) — наш уникальный ход

У нас уже есть `hh eval` и `hh gate`. Деньги резервируются при pull, списываются **только когда quality gate прошёл на машине покупателя**.

- **Момент оплаты:** прохождение gate (списание), pull (резерв).
- **Защита:** снимает главный барьер покупателя — недоверие. Так не делает никто на рынке.
- **Что строить:** эскроу-логика + подписанный результат gate (анти-фрод: покупатель может соврать, что gate не прошёл — нужен signed receipt по образцу agentic.market).

### M4. Bounties — заказная разработка харнесов

Покупатель постит задачу с бюджетом, авторы откликаются, приёмка через evals. Самые быстрые деньги для соло-автора сегодня (аналог: GPT-консалтинг $5–20k за проект против $100–500/мес в GPT Store).

- **Момент оплаты:** эскроу при старте, релиз при прохождении приёмочных evals.

### M5. B2B приватный реестр

Приватный контур для команды: свои харнесы, контроль версий, аудит пермишенов, SSO. Скучно, но это самые надёжные деньги платформы ($/seat или flat $99–499/мес).

- **Момент оплаты:** месячная подписка компании.

## 2. Платёжные рельсы: два канала параллельно

**x402 (крипто, agent-native):**
```
hh pull premium-harness
  → API отвечает 402 + PaymentRequired (сумма, адрес, chain)
  → кошелёк агента/пользователя платит USDC (Base/Solana)
  → facilitator подтверждает → отдаём архив/токен доступа
```
Один HTTP-раундтрип, без аккаунтов, комиссия ~0, работает для автономных агентов (наш llms.txt-сценарий: агент сам нашёл и сам купил). Практический плюс: не требует Stripe-энтити — только кошелёк.

**Фиат (Merchant of Record — Paddle/LemonSqueezy):** для корпоратов и M5 — им нужны инвойсы и VAT-комплаенс, не USDC. MCPize берёт на себя налоги в 135+ валютах — это часть их 20%; MoR решает то же за ~5%+.

Реальность x402 (важно для питча без розовых очков): 165M+ транзакций и $50M объёма к апрелю 2026, но падение дневного объёма на 92% с декабрьского пика и вывод CoinDesk «спрос на микроплатежи ещё не пришёл». Рельсы строим, спрос на них не закладываем в базовый сценарий.

## 3. Защита от воровства — слоями

| Слой | Механика | От чего защищает |
|---|---|---|
| Экономический | M1–M3: ценность в исполнении/обновлениях/гарантии | Главный слой: копия бесполезна |
| Entitlement | Платный архив отдаётся только авторизованному покупателю/кошельку | Массовое скачивание |
| Watermark | Невидимая маркировка копии per-buyer | Трейсабилити перепродажи |
| Дубликат-детектор | `hh diff` при publish: клон чужого харнеса не публикуется и не получает trust-сигналов | Перепродажа внутри нашего реестра |
| Социальный | Звёзды, треды, история запусков, verified author не копируются с файлом | Краденая копия мертва, как форк без истории |

Честно: от копирования «для себя» не защититься ничем. Защищаемся от перепродажи и утечки ценности — весь SaaS живёт так же.

## 4. Конкурентный ландшафт (июль 2026)

Рынок разложился на 4 слоя:

### Слой 1. Продажа файлов (наши прямые конкуренты по «продаже скилов»)

**Agensi** (agensi.io) — маркетплейс SKILL.md для 20+ кодинг-агентов (Claude Code, Cursor, Codex CLI...). Покупка → получаешь файл → ставишь руками. Комиссия 30%, выплаты Stripe Connect, 8-точечный секьюрити-скан перед листингом. Топ-скилы $500–3000/мес, медиана <$50/мес. **Слабости:** продают plaintext без runtime-защиты, нет evals/подтверждения качества, нет agent-native покупки, нет социального слоя.

**PromptBase** — крупнейший файловый маркетплейс промптов (310k+ промптов, $1.99–9.99 за штуку, автор получает 80%). Новички $50–200/мес, стабильные селлеры $500–1500/мес, топы $2–5k/мес. Рынок промпт-маркетплейсов оценивают в $2.5B (2026) с прогнозом $7B к 2030. **Урок:** файловая модель живёт, но потолок низкий и медиана бедная.

### Слой 2. Продажа исполнения (доказанная экономика — наш бенчмарк)

**Apify Store** — маркетплейс скраперов/акторов: разработчик публикует, покупатель платит за использование. 80/20, выплачено $1M+ разработчикам за один месяц, топ-авторы $10k+/мес MRR. С октября 2026 полностью переходят на pay-per-event/usage (рентная модель убита; авторы без миграции теряют 40–70% выручки). **Урок:** зрелые платформы приходят к usage-based; продаётся исполнение, не файл.

**MCPize** (запуск начала 2026) — единственный MCP-маркетплейс с revenue share: 80/20 (фаундер-мемберы 85%), берёт на себя хостинг, SSL, Stripe, налоги в 135+ валютах, саппорт. Модели: подписка ($9–199/мес), one-time, usage-based через x402 (USDC на Base). Выплаты 1-го числа, минимум $100. **Слабости:** только MCP-серверы, нет скилов/воркфлоу/харнесов, нет evals, нет социума.

**GPT Store** — рев-шер по engagement (~$0.03/диалог): большинство авторов упёрлись в $100–500/мес, топ-0.01% выше; авторы массово ушли в B2B-консалтинг ($5–20k за внедрение). **Урок:** usage rev-share без ценовой власти автора = бедность; автор должен сам назначать цену.

### Слой 3. Agent-native рельсы и каталоги (Coinbase-мир)

**agentic.market + x402 Bazaar (Coinbase)** — «app store для агентов»: каталог 1536+ x402-сервисов (inference, data, search, media, trading), цены $0.001–7.50 за вызов в USDC, без аккаунтов и ключей, discovery через llms.txt и MCP, самоиндексация по живым платежам, подписанные receipts на каждую транзакцию. **Продают API-вызовы, не харнесы/скилы** — это другой слой продукта. Для нас это скорее рельсы и валидация agent-native покупок, чем конкурент. **Но:** у них уже есть `npx skills add coinbase/agentic-wallet-skills` — в дистрибуцию скилов они одной ногой зашли. Риск отслеживаем.

### Слой 4. Бесплатные реестры (конкуренты за внимание, не за деньги)

**Smithery / mcp.so / PulseMCP** — топ-3 MCP-хаба (у Smithery 7000+ серверов, CLI, хостинг). Монетизация: платные тарифы за хостинг для самих авторов ($30/мес), **автору не платят ничего**. **Anthropic plugin marketplaces** — официальная git-based дистрибуция, бесплатно, механизма монетизации нет. **tonsofskills.com / claudemarketplaces.com** — каталоги-агрегаторы (425 плагинов, 2810 скилов), бесплатно. **Урок:** discovery уже коммодитизирован — конкурировать одним каталогом нельзя; Hugging Face Spaces, LangChain Hub, Vercel Gallery — та же история (free distribution, заработок только косвенный: лиды на консалтинг).

## 6. Выводы и последовательность

1. **Не начинать с продажи файлов** — это рынок Agensi с медианой <$50/мес. Наша экономика — слои 2–3: исполнение и agent-native.
2. **Sequencing:** (1) бесплатная дистрибуция + trust → (2) bounties и подписки на обновления для первых 10–20 авторов вручную → (3) x402-рельсы, платные endpoints, gate-эскроу → (4) B2B приватные реестры.
3. **Комиссия:** входим с 10–15% (агрессивнее Agensi 30% и Apify/MCPize 20%) — на холодном старте маржа платформы не важна, важна плотность авторов.
4. **Бенчмарк успеха для питча авторам:** «топ-автор Apify делает $10k+/мес на исполнении; топ-скил на файловом маркетплейсе — $3k/мес; наша модель ближе к Apify».
5. **Главный платформенный риск** — Anthropic включит монетизацию в официальные marketplaces. Ответ: мульти-CLI нейтральность (как Agensi: 20+ агентов), agent-native покупка и outcome-based модель, которые платформа-вендор быстро не сделает.
6. **Coinbase — партнёр по рельсам, конкурент по дистрибуции скилов** (у них уже `npx skills add`). Правильный ход: интегрироваться с x402 Bazaar раньше, чем они дорастут до харнесов, и попасть в их каталог как сервис.
